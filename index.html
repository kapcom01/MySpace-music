🎵 Music Discovery Hub

Extract song metadata from MySpace profiles and discover the songs on YouTube.

Step 1 — Provide a MySpace profile URL

Example: https://myspace.com/myspace/music/songs

Why might this fail?
  • The profile URL may be private, deleted, or mistyped.
  • Public CORS proxies (allorigins, corsproxy) occasionally go down or rate-limit.
  • MySpace itself may be temporarily unreachable.
  • Try again in a minute, or try a different MySpace profile URL.

Step 2 — Discovered songs

    About this tool

    • 🔒 Privacy-first: all processing happens in your browser.
    • 🎵 Each song has a YouTube search button and a Download button.
    • ⬇️ The Download button opens the song's direct audio URL when the profile exposes one (via <audio src> or <source src>); otherwise it opens the MySpace profile's music page.
    • ⚖️ The user is responsible for ensuring any music downloaded is royalty-free or otherwise licensed for download. A license checker is planned.
    • 🌐 Uses public CORS proxies (allorigins.win, corsproxy.io) with automatic fallback.
